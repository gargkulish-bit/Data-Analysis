# Data-Analysis
Power BI Data analysis dashboard project.
This project presents an AI-driven educational analytics system designed for learner segmentation and personalized course recommendation within online learning environments. The system combines exploratory data analysis, feature engineering, clustering techniques, ensemble machine learning models, and recommendation logic to analyze learner behaviour and predict course-level preferences.

The project was developed under the Unified Mentor educational analytics project framework as part of an undergraduate applied machine learning and educational data science study.
Dataset Information
A statistically realistic synthetic educational dataset was used to simulate:

Learner demographics
Course interactions
Transaction behaviour
Instructor profiles
Course ratings and engagement patterns
The dataset was designed to preserve meaningful educational relationships while addressing privacy and accessibility limitations associated with real-world educational data.
